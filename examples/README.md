# Examples

Real `factory` runs, kept as evidence of how the pipeline behaves end to end.

## stock_guidance_kr

A personal Korean-stock guidance tool. Full Stage 0–4 design output (idea
gate, blueprint, PRD, architecture, code design, verification plan, ADRs)
plus the final `SUMMARY.md` with build/test evidence from Stage 5
implementation.

- `stage0_profile.json` — idea concreteness score + profile
- `stage1_blueprint.json` / `stage1_history.json` — TCI loop history
- `stage2_prd.json`, `stage3_design.json`, `stage4_verify.json`
- `adr/` — individual decision records (ADR-001..005)
- `consistency_*.json` — cross-stage consistency audits
- `SUMMARY.md` — final summary with verification evidence
